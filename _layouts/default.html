about me

For you that don't know me, my name's Andrée Hansson and live in Sweden, I work at presis! as a Front-end Developer, I've also done a few presentations (both internally and externally) on front-end development. I created the 960 Gridder bookmarklet, which helps you implement designs made with the 960.gs grid system very easily. I have also contributed to the jQuery project, and I'm currently working on the new project-wide re-design for jQuery. I have also had the privilege to work with Addy Osmani as a technical reviewer on some of his great articles and posts. For more information and other fun stuff, you can check out my Twitter and GitHub pages.

This site was thrown together using jekyll, theme implemented from scratch but is heavily influenced by Minimum theme.
